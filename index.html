What I Learned

At first, I felt like my understanding of websites was pretty basic. I knew about the "WWW" (World Wide Web) and how it let us communicate with people all over the world in seconds. I learned that HTML (Hypertext Markup Language) is the main code used to structure a webpage, and that browsers download it to display the content.

My Basic understanding of Code:

< element + attribute = “action” >



  • Element: This is the HTML tag that defines what kind of content or structure you want. For example, a div creates a section, and an img tag adds an image.
  • Attribute: These give extra info about the element. For example, a class groups multiple elements together, and an id helps you pick out a specific element. Attributes can also define things like styles or links to other resources.
  • Action: This is where things happen. In HTML, “action” refers to the attributes that control behavior, like href for links or src for images. In CSS and JavaScript, actions are things like "change color," or "move this element."


All of these actions in HTML help get the content you want onto your webpage. But when it comes to things like alignment and color, that’s where CSS (Cascading Style Sheets) comes in. CSS is a separate file that’s all about styling your HTML. < link rel="stylesheet" href="styles.css" > In short, HTML is what builds the structure of a webpage, while CSS makes it look good. Learning to code is about figuring out how to put elements and attributes together, and then using CSS to style them.

Bio

After taking a 10-year sabbatical from school, I returned and earned an AA from SCC. I am on my way to earning a BA in Studio Art from Eastern Washington University. I am an analog artist who has found themselves in a digital realm. Having years of experience with painting and drawing I decided to challenge myself once again. This time I went virtual with my creative ability, and I was not disappointed in the results. My personal experience with computers extended to Adobe and Microsoft programs. That was however at the start of this quarter, Fall 2024. Now I have dabbled in multiple animation software and created web pages through code.

Reflection

Hand me a paintbrush, hand me a sketchpad, perhaps I can give you a quick rundown on the evolution of art, from medieval times to the contemporary era. If it comes to explaining my coding process, I could not do that. What I can tell you is that I do know how to code. At some point this quarter, everything just clicked. I would open a tag and automatically add a closing tag without thinking. I found myself troubleshooting my code with confidence. Over time, I developed my own style in my projects. By the end of it, I successfully built functional interactive websites.
